\imsd\infinityUser\common

\imsd\infinityUser\commonORMBase

ORM公用基类.

在ActiveRecord基础上扩展一些功能.
扩展功能.

  • 表前缀。通过类静态成员：$tbPrefix. 自动加入前缀
  • 统一提交方法，传入回调，内所有操作都会被合并一个事务执行
  • 时间字段自动设置，构造时补全created_at 提交时补全updated_at

tableName()

tableName(null|string  $otherTable = null) : string|null

重载yii,为本系统的表增加前缀：iu.

此类是所有ORM类都会继承的，因此可以实现所有表加前缀
加前缀目的与其他系统/模块区分开
yii只能在数据库链接上全局加前缀，所以会重载此方法自定义
